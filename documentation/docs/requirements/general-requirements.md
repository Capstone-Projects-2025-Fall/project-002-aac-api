# General Requirements
## Hardware Requirements
- Microphone Access
- Laptop, Computer, or any device to play the AAC games
- Internet Access

## Software Requirements

<b>Server-Side(Backend):</b>
- Python (Machine Learning, speech recognition)
- Node.js (Audio processing, API management)
- AWS (hosting API, speech input recognition)

## Project Management Tools
- GitHub (For Version control and repository)
- Jira (Task management and tracking)
