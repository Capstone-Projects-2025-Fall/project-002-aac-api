# Development Environment

An overview of the development environment to work with the AAC API project

## Required Hardware
- A computer running **Windows**, **macOS**, or **Linux**.
- At least **4GB of RAM** available
- **500MB of free disk space**
- A stable internet connection

## Tools

- Node.js (v18 ot later)
    - Runs the Express server and manages JavaScript dependencies.
- Python
    - Python runs the speech recognition script
- Git
    - Git is used for vesrion control
- Visual Studio Code
    - Preferred code editor for git integration and extensions for productivity
