# Minimal Example

This is a minimal example demonstrating how to integrate the AAC SDK with a game.

## Features

- Basic AACClient initialization
- Intent event handling
- Simple game state management
- Voice command processing

## Usage

```bash
# Build the SDK first
npm run build

# Run the example
npx ts-node examples/minimal/index.ts
```

## Game Commands

Try saying these voice commands:
- "move forward" / "move backward" / "move left" / "move right"
- "jump"
- "attack"
- "stop"
- "interact"

## Code Structure

- `index.ts` - Main game class with AAC SDK integration
